Choosing a component

If you need…UseNot
A standard action in a layout or form Button FAB — reserved for the single prominent page-level action
Several independent on/off choices Checkbox Radio — radios are one-of-many, not many-of-many
A setting that takes effect immediately Toggle Checkbox — checkboxes imply a submit step
A read-only status or category label Pill Selection Pill — that one is interactive (filters, multi-select)
An identity (photo / initials / icon) Avatar Raw <img> — Avatar handles shape, fallbacks, and status
Identity + name + meta in one row Title Block Hand-rolled Avatar + text — Title Block fixes the spacing and truncation
Mobile options/pickers from the bottom edge Bottom Sheet A dropdown on mobile — sheets are the thumb-reach pattern; native apps use platform containers
A tappable or selectable list row Card A bare Title Block in a clickable div — Card handles the control semantics, states and hit area
