Component

Avatar

Beta v1.0.0

Represents a child profile identity — photo, initials, icon, or add placeholder. Default size is 64px but scales to fit parent containers. The status dot flags unchecked profile data, not online presence.

3Usage guidelines

✓ Do

  • Use for child profile identity in lists, headers, and pickers
  • Show initials when no photo is available
  • Use icon-outline for empty "add profile" slots
  • Set hasStatus=true for unchecked / new profile data
  • Use showStackedAvatar to indicate a linked secondary profile
  • Always provide accessibilityLabel with the full person name

✗ Don't

  • Use for brand logos, app icons, or decorative illustration → use Icon
  • Treat the status dot as an online/offline presence indicator
  • Use for non-person entities (rooms, classes, categories)
  • Override icon-colour icon fills — let the icon's embedded palette stand
  • Use initials alone as the accessible name

Status dot semantics

Not a presence indicator. hasStatus=true means the child profile has unchecked or new data awaiting review — not an online/offline/busy state. If you need presence, use a dedicated pattern.

Content guidelines

5Accessibility

🏷️
Accessible name — required on every instance
Use the full person name ("Jane Appleseed"), never just initials. Initials are visual shorthand only.
🔔
Status dot — don't rely on colour alone
When hasStatus=true, include "has unchecked data" or equivalent in the accessible name or via aria-describedby.
👥
Stacked badge — name both profiles
When showStackedAvatar=true, convey both: "Jane Appleseed, linked to Lisa Brown".
➕
icon-outline (add) — label as action
Label the action intent ("Add child profile"), not the icon ("Plus icon"). Minimum 44×44pt touch target when interactive.
🖼️
image — alt text
If the photo is decorative in context, alt="" + label on the container. If it conveys identity independently, use a meaningful alt matching the accessible name.
<!-- All types: container carries the accessible name -->
<div role="img" aria-label="Jane Appleseed">

</div>

<div role="img" aria-label="Jane Appleseed, has unchecked data">…</div>

<div role="img" aria-label="Jane Appleseed, linked to Lisa Brown">…</div>

<!-- icon-outline as interactive add -->
<button type="button" aria-label="Add child profile">

import { computed, withDefaults, defineProps } from 'vue';

type AvatarType  = 'image' | 'initials' | 'icon-colour' | 'icon-outline';
type AvatarShape = 'round' | 'square';

const props = withDefaults(defineProps<{
  type?:              AvatarType;
  shape?:             AvatarShape;
  size?:              number;
  src?:               string;
  initials?:          string;
  iconSrc?:           string;
  hasStatus?:         boolean;
  showStackedAvatar?: boolean;
  stackedInitials?:   string;
  accessibilityLabel: string;
  interactive?:       boolean;
}>(), {
  type: 'icon-colour', shape: 'round', size: 64,
  hasStatus: false, showStackedAvatar: false, interactive: false,
});

const showBorder = computed(() =>
  props.shape === 'round' ||
  (props.shape === 'square' && props.type === 'icon-colour')
);

const ariaLabel = computed(() => {
  let label = props.accessibilityLabel;
  if (props.hasStatus)         label += ', has unchecked data';
  if (props.showStackedAvatar) label += `, linked to ${props.stackedInitials ?? 'secondary profile'}`;
  return label;
});
</script>

SwiftUI 
import SwiftUI

enum AvatarType  { case image, initials, iconColour, iconOutline }
enum AvatarShape { case round, square }

struct Avatar: View {
    var type:               AvatarType  = .iconColour
    var shape:              AvatarShape = .round
    var size:               CGFloat     = 64
    var imageURL:           URL?        = nil
    var initials:           String?     = nil
    var icon:               Image?      = nil
    var hasStatus:          Bool        = false
    var showStackedAvatar:  Bool        = false
    var stackedInitials:    String?     = nil
    var accessibilityLabel: String

    // Colours resolved from Stardust tokens
    private let primaryGreen   = Color(red: 0,     green: 0.467, blue: 0.420) // colour/action/primary
    private let surfaceGreen   = Color(red: 0.945, green: 0.973, blue: 0.945) // colour/surface/green
    private let surfaceGrey    = Color(red: 0.965, green: 0.965, blue: 0.965) // colour/surface/grey
    private let textPrimary    = Color(red: 0.145, green: 0.180, blue: 0.204) // colour/text/primary
    private let successDefault = Color(red: 0.004, green: 0.639, blue: 0.616) // colour/feedback/success/default

    private var clipShape: AnyShape {
        shape == .round ? AnyShape(Circle()) : AnyShape(RoundedRectangle(cornerRadius: 8))
    }
    private var showBorder: Bool {
        shape == .round || (shape == .square && type == .iconColour)
    }

    var body: some View {
        ZStack(alignment: .topTrailing) {
            // ① Content — clipped to shape
            avatarContent
                .frame(width: size, height: size)
                .clipShape(clipShape)
                .overlay(showBorder ?
                    clipShape.stroke(Color.white, lineWidth: 2) : nil
                )

            // ③ Status dot — outside clip, top-right
            if hasStatus {
                Circle()
                    .fill(successDefault)
                    .frame(width: size * 0.25, height: size * 0.25)
                    .overlay(Circle().stroke(Color.white, lineWidth: 2))
                    .offset(x: 2, y: -2)
            }
        }
        .overlay(alignment: .bottomTrailing) {
            // ④ Stacked badge — outside clip, bottom-right
            if showStackedAvatar, let init = stackedInitials {
                Text(init)
                    .font(.system(size: size * 0.375))
                    .foregroundColor(.white)
                    .frame(width: size * 0.667, height: size * 0.667)
                    .background(textPrimary)
                    .clipShape(Circle())
                    .overlay(Circle().stroke(Color.white, lineWidth: 2))
                    .offset(x: size * 0.1, y: size * 0.1)
            }
        }
        .accessibilityElement(children: .ignore)
        .accessibilityLabel(a11yLabel)
    }

    private var a11yLabel: String {
        var s = accessibilityLabel
        if hasStatus         { s += ", has unchecked data" }
        if showStackedAvatar { s += ", linked to \(stackedInitials ?? "secondary profile")" }
        return s
    }

    @ViewBuilder
    private var avatarContent: some View {
        switch type {
        case .image:
            AsyncImage(url: imageURL) { img in
                img.resizable().scaledToFill()
            } placeholder: {
                ZStack { surfaceGreen; Text(initials ?? "")
                    .font(.system(size: size * 0.5)).foregroundColor(primaryGreen) }
            }
        case .initials:
            ZStack { surfaceGreen
                Text(initials ?? "").font(.system(size: size * 0.5)).foregroundColor(primaryGreen) }
        case .iconColour:
            ZStack { surfaceGrey
                icon?.resizable().scaledToFit().frame(width: size * 0.5, height: size * 0.5) }
        case .iconOutline:
            ZStack { primaryGreen
                icon?.resizable().scaledToFit()
                    .frame(width: size * 0.5, height: size * 0.5)
                    .colorMultiply(.white) }
        }
    }
}

Jetpack Compose (Kotlin) 
import androidx.compose.foundation.background
import androidx.compose.foundation.border
import androidx.compose.foundation.layout.*
import androidx.compose.foundation.shape.CircleShape
import androidx.compose.foundation.shape.RoundedCornerShape
import androidx.compose.material3.Text
import androidx.compose.runtime.Composable
import androidx.compose.ui.Alignment
import androidx.compose.ui.Modifier
import androidx.compose.ui.draw.clip
import androidx.compose.ui.graphics.Color
import androidx.compose.ui.layout.ContentScale
import androidx.compose.ui.semantics.contentDescription
import androidx.compose.ui.semantics.semantics
import androidx.compose.ui.unit.Dp
import androidx.compose.ui.unit.dp
import androidx.compose.ui.unit.sp
import coil.compose.AsyncImage

enum class AvatarType  { Image, Initials, IconColour, IconOutline }
enum class AvatarShape { Round, Square }

// Colours resolved from Stardust tokens
private val PrimaryGreen   = Color(0xFF00776B) // colour/action/primary
private val SurfaceGreen   = Color(0xFFF1F8F2) // colour/surface/green
private val SurfaceGrey    = Color(0xFFF6F6F6) // colour/surface/grey
private val TextPrimary    = Color(0xFF252525) // colour/text/primary
private val SuccessDefault = Color(0xFF01A39D) // colour/feedback/success/default

@Composable
fun Avatar(
    accessibilityLabel: String,
    modifier:           Modifier    = Modifier,
    type:               AvatarType  = AvatarType.IconColour,
    shape:              AvatarShape = AvatarShape.Round,
    size:               Dp          = 64.dp,
    imageUrl:           String?     = null,
    initials:           String?     = null,
    icon:               @Composable (() -> Unit)? = null,
    hasStatus:          Boolean     = false,
    showStackedAvatar:  Boolean     = false,
    stackedInitials:    String?     = null,
) {
    val clipShape  = if (shape == AvatarShape.Round) CircleShape
                     else RoundedCornerShape(8.dp)
    val showBorder = shape == AvatarShape.Round ||
                     (shape == AvatarShape.Square && type == AvatarType.IconColour)
    val a11yLabel  = buildString {
        append(accessibilityLabel)
        if (hasStatus)         append(", has unchecked data")
        if (showStackedAvatar) append(", linked to ${stackedInitials ?: "secondary profile"}")
    }

    Box(
        modifier = modifier
            .size(size)
            .semantics { contentDescription = a11yLabel }
    ) {
        // ① Content — clipped to shape
        Box(
            modifier = Modifier
                .size(size)
                .clip(clipShape)
                .then(if (showBorder) Modifier.border(2.dp, Color.White, clipShape)
                      else Modifier),
            contentAlignment = Alignment.Center
        ) {
            when (type) {
                AvatarType.Image -> AsyncImage(
                    model = imageUrl, contentDescription = null,
                    contentScale = ContentScale.Crop,
                    modifier = Modifier.fillMaxSize()
                )
                AvatarType.Initials -> Box(
                    Modifier.fillMaxSize().background(SurfaceGreen),
                    contentAlignment = Alignment.Center
                ) {
                    Text(initials ?: "", color = PrimaryGreen,
                         fontSize = (size.value * 0.5f).sp)
                }
                AvatarType.IconColour -> Box(
                    Modifier.fillMaxSize().background(SurfaceGrey),
                    contentAlignment = Alignment.Center
                ) { icon?.invoke() }
                AvatarType.IconOutline -> Box(
                    Modifier.fillMaxSize().background(PrimaryGreen),
                    contentAlignment = Alignment.Center
                ) { icon?.invoke() } // Caller must tint icon to Color.White
            }
        }

        // ③ Status dot — outside clip, top-right
        if (hasStatus) {
            Box(
                modifier = Modifier
                    .size(size * 0.25f)
                    .clip(CircleShape)
                    .background(SuccessDefault)
                    .border(2.dp, Color.White, CircleShape)
                    .align(Alignment.TopEnd)
                    .offset(x = 2.dp, y = (-2).dp)
            )
        }

        // ④ Stacked badge — outside clip, bottom-right
        if (showStackedAvatar && stackedInitials != null) {
            Box(
                modifier = Modifier
                    .size(size * 0.667f)
                    .clip(CircleShape)
                    .background(TextPrimary)
                    .border(2.dp, Color.White, CircleShape)
                    .align(Alignment.BottomEnd)
                    .offset(x = size * 0.1f, y = size * 0.1f),
                contentAlignment = Alignment.Center
            ) {
                Text(stackedInitials, color = Color.White,
                     fontSize = (size.value * 0.375f).sp)
            }
        }
    }
}

Acceptance criteria

  • All 4 types × 2 shapes render correctly on web, iOS, and Android
  • Avatar size controlled via single value — all child elements scale proportionally
  • White 2px border ring on all round types; square border on icon-colour only
  • hasStatus shows teal dot with white ring at top-right; hidden when false
  • showStackedAvatar shows dark badge at bottom-right; hidden when false
  • Status dot and stacked badge visible beyond container edge — not clipped
  • Icon slots accept any iconColoured or iconOutline SVG
  • icon-outline icon renders white on teal background
  • accessibilityLabel always set to full person name; status and badge reflected in label
  • Interactive avatars meet 44px / 44pt / 48dp minimum touch target
  • Photo type uses object-cover fill within the avatar bounds

7Open Questions

Design decisions worth confirming before engineering finalises implementation.

⚠️
1 · Square stacked badge positioning The spec defines square stacked badge at top: 30.67px, left: 30.67px on a 64px container, but this varies at other sizes. Confirm whether the badge should always be anchored bottom-right at a fixed overhang percentage, or use the absolute values from Figma and scale manually.
⚠️
2 · Photo fallback order When a photo URL fails to load, should the component fall back to initials, icon-colour, or show a broken-image state? Confirm the expected fallback chain before implementing the image type.
ℹ️
3 · Stacked badge content beyond initials The current spec only defines initials in the stacked badge. Confirm whether the badge ever needs to show a photo or icon instead, which would require a nested Avatar component inside the badge.
