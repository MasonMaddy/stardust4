Grey — default · neutral

For neutral tips, notes, or general guidance that carries no semantic status.

Green — success

Confirms an action completed successfully — saved, submitted, signed in, or published.

Orange — warning

Signals a potential problem or something that needs attention before continuing.

Red — error

Communicates a failure or blocking error that prevents the user from proceeding.

Purple — info

Provides contextual information or a helpful tip — not an error or warning.
