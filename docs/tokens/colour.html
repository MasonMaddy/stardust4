Colour

Synced from Figma 2026-06-02 · 2 collections: base + mapped

Stardust uses a two-tier colour model. Primitives (base collection) are raw hex values named by hue and shade. Semantic tokens (mapped collection) are intent-based aliases that reference primitives — these are what components use.
