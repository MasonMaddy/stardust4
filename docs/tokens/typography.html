Typography

Font: Inter · Synced from Figma 2026-06-02

Stardust uses Inter as the single font family for all product text. Size, line-height, and weight are separate primitive tokens combined into named text styles. iOS and Android use their native system fonts at the equivalent sizes — see the platform mapping below.

Platform mapping

ℹ️
Native platforms use their system font at equivalent sizes. iOS uses SF Pro (dynamic type). Android uses Roboto (Material 3 typography scale). The token name is the shared contract across all three platforms — engineers use the platform-specific style that matches the token size and weight.

⚠️
iOS Dynamic Type SF Pro styles scale with the user's preferred text size setting. Always support Dynamic Type on iOS — never hardcode point sizes in UIKit/SwiftUI. Map Stardust token sizes to .font() modifier with the semantic style as the starting point, and let the system scale from there.
