Token Foundation

Typography

Font: Inter (all text) · Synced from Figma 2026-06-02

Stardust uses Inter as the single font family across all component text. Font size, line-height, and weight are defined as separate primitive tokens — combined into text style tokens at the component level.

Font families

Aa
Inter — body & UI
Used for all Stardust component text: labels, body copy, captions.
Token: --sd-font-family · Figma: font-family/Inter
ℹ️
Site chrome uses Sora This documentation site uses Sora for headings (via --font-heading in main.css). Sora is not a Stardust component token — it's the Xplor corporate brand font for marketing and site chrome only. Do not use it in product components.
