Token Foundation

Spacing

Base unit: 4px · Synced from Figma 2026-06-02

All spacing is built on a 4px base grid. The primitive scale uses numeric steps. Semantic tokens use descriptive names (sm, md, lg) for component padding, and intent names (tight, default, loose, wide) for layout gaps.
