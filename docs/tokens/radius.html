Radius

4 steps: sm · m · lg · full · Synced from Figma 2026-06-02

Usage

ℹ️
Naming rule Radius uses t-shirt sizing only (sm, m, lg, full). Do not use numeric names like radius-8 or words like radius-medium. Note: the medium step uses m (not md) to match the Figma token name.
