Token Foundation

Radius

4 steps: sm · m · lg · full · Synced from Figma 2026-06-02

Border-radius tokens use t-shirt sizing. The scale ranges from sm (4px) for subtle rounding through full (9999px) for pills and FABs. All tokens are defined in the base collection and used directly (no semantic alias layer for radius).

Usage

ℹ️
Naming rule Radius uses t-shirt sizing (sm, m, lg, full). Do not use numeric or descriptive names like radius-8 or radius-medium. The exception is full which is a semantic name for pill/circle shapes.
